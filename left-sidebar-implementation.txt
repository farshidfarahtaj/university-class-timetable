# Collapsible Left Sidebar Implementation Guide

This document explains how to implement the collapsible left sidebar component in your PHP files.

## 1. Files Created/Modified

- `LeftSidebar.php`: The reusable sidebar component with toggle functionality
- `left-sidebar.css`: CSS styles for the sidebar including animations
- Updated `scripts.js`: Added toggle functionality and keyboard support
- Updated examples: `index.php`, `admin_dashboard.php`, and `student_dashboard.php`

## 2. How to Implement in Other Pages

For each page you want to update, follow these steps:

1. Add the CSS link in your `<head>` section:
```html
<link rel="stylesheet" href="left-sidebar.css">
```

2. Remove the old sidebar code:
```html
<!-- Remove this old sidebar code -->
<div id="mySidepanel" class="sidepanel">
    <a href="javascript:void(0)" class="closebtn" onclick="closeNav()">&times;</a>
    <!-- your links -->
</div>
<button class="openbtn" onclick="openNav()">&#9776; Menu</button>
```

3. Add the new sidebar code right after the `<body>` tag:
```php
<?php
// Make sure session is started
if (session_status() === PHP_SESSION_NONE) {
    session_start();
}
// Include the left sidebar component
include 'LeftSidebar.php';
?>
```

4. Change the `container` class to `main-content`:
```html
<!-- Replace this -->
<div class="container">

<!-- With this -->
<div class="main-content">
```

5. Keep all the existing content within the `main-content` div.

6. Make sure the closing `</div>` is right before the closing `</body>` tag.

## 4. Features of the Collapsible Sidebar

- The sidebar is hidden by default and appears when the hamburger menu button is clicked
- Smooth sliding animation when opening/closing
- Responsive design that adapts to different screen sizes
- Close button (✕) inside the sidebar
- Keyboard support (press Escape to close)
- On large screens (≥992px), the sidebar pushes content when opened
- On smaller screens, it overlays content

## 5. Customization Options

You can easily customize the sidebar by modifying the CSS variables at the top of `left-sidebar.css`:

```css
:root {
    --sidebar-bg: #003366;       /* Background color */
    --sidebar-width: 250px;      /* Width of sidebar */
    --sidebar-text: #ffffff;     /* Text color */
    --sidebar-highlight: #ffcc00; /* Highlight color for hover/active */
    --transition-speed: 0.3s;    /* Animation speed */
}
```

For any issues or questions, refer to the example files for implementation details.
